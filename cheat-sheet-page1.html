Output

puts

puts "This will print the text to the screen then move to the next line"

print

print "This will print the text to the screen without moving to the next line"

Comments

# This is a comment
puts "This runs"

# Ruby does not have a true multi-line comment.
# Use # at the start of each line.

Variables

Local Variables

  • Start with a lowercase letter or underscore, use snake_case.
  • Scope is limited to the block or method where it is defined.
  • Created when first assigned a value (no prior declaration needed).
  • Be descriptive, e.g. total_monthly_sales not total
  • Avoid reserved words, e.g. if, elsif, else, end, class, def, etc.
count = 10
_name = "Ruby"
total_price = count * 5

Constants

  • Uppercase names for values intended not to change
MAX_ITEMS = 10

Basic Data Types

# String: text data
"Hello, World!"

# Integer: whole numbers
42
-7

# Float: decimal numbers
3.14
-0.001

# Boolean: true or false
true
false

# Nil: represents no value
nil

Arrays

  • Ordered list of items, indexed by position (starts at 0)
# Create an array
numbers = [1, 2, 3]

# Access elements (index starts at 0)
numbers[0]   # 1
numbers[1]   # 2

# Add elements
numbers << 4        # append using <<
numbers.push(5)     # append using method

# Remove elements
numbers.pop         # removes last item

Hashes

  • Keys are usually symbols or strings
# Create a hash
user = {
  "name" => "Alice",
  "age"  => 25
}

# Access values
user["name"]   # "Alice"
user["age"]    # 25

# Add a key-value pair
user["email"] = "alice@example.com"

# Remove a key-value pair
user.delete("age")

Symbols (:symbol)

  • Lightweight identifiers used as labels
  • Common as hash keys and method arguments
:status

Conditionals

if, elsif, else

# Basic if statement
if total > 40
    puts "Large order"
end

# Ruby-style one line
puts "Large order" if total > 40

# Using elsif and else
if total > 100
  puts "Free shipping"
elsif total > 40
  puts "Standard shipping"
else
  puts "Shipping applies"
end

Case / When (conditional branching)

  • Used to compare one value against many options
  • Cleaner than long if / elsif chains
  • Uses when instead of elsif
status = :paid    # value being checked

case status
when :paid
  puts "Order paid"
when :pending
  puts "Awaiting payment"
else
  puts "Unknown status"
end

Comparison & Logical Operators

  • Used to compare values and in loops
  • Return true or false
a = 10
b = 5

a == b     # false
a != b     # true
a > b      # true
a <= b     # false

a > 5 && b < 10   # true
a < 5 || b < 10   # true
!false            # true

Loops/Iteration

  • Each: Iterates over an array:
orders.each do |order|
    puts order[:id]
end
  • Times: Repeats a block a specified number of times:
5.times do |i|
    puts "Iteration #{i + 1}"
end

Common Enumerable Methods

  • Used with arrays and collections
  • Return new values, do not modify originals
numbers = [1, 2, 3, 4, 5]

numbers.map { |n| n * 2 }        # [2, 4, 6, 8, 10]
numbers.select { |n| n.even? }   # [2, 4]
numbers.reject { |n| n < 3 }     # [3, 4, 5]
numbers.find { |n| n > 3 }       # 4

Methods

  • Reusable blocks of code that perform a task
  • Can take arguments (parameters)
  • The last evaluated expression is returned automatically (implicit return)
  • return is optional but exits the method immediately
  • Parentheses are optional when calling methods with no arguments
def total(price, qty)
    price * qty
end
