Ruby and Rails structure

High-level ideas that explain how Ruby and Rails applications are organised.

  • Object Orientation: Ruby is object-oriented Everything is an object, including numbers, strings and arrays.
  • Rails: A web application framework written in Ruby that provides structure, conventions and tools for building web apps quickly.
  • MVC (Model View Controller): The architectural pattern used by Rails to separate data, UI and application logic.
